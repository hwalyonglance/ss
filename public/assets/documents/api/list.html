API reference for Angular Material list

import {MdListModule} from '@angular/material';

Directives

MdListDivider

selector: md-divider

MdList

selector: md-list md-nav-list
Properties
NameDescription
@Input()

disableRipple

Whether the ripple effect should be disabled on the list-items or not. This flag only has an effect for `md-nav-list` components.

MdListItem

selector: md-list-item a[md-list-item]
Properties
NameDescription
@Input()

disableRipple

Whether the ripple effect on click should be disabled. This applies only to list items that are part of a nav list. The value of `disableRipple` on the `md-nav-list` overrides this flag.
Methods
isRippleEnabled
Whether this list item should show a ripple effect when clicked.
